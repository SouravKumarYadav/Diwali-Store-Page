# Diwali-Store-Page

![Diwali-Store-Page Preview](https://github.com/SouravKumarYadav/Diwali-Store-Page/blob/main/Diwali%20Store%20Page.png)

> A vibrant Diwali store page showcasing festive products, built with HTML, CSS, and Bootstrap for clean alignment and responsive design.

## 🚀 Live Demo  
👉 [Click here to view the live site](https://souravkumaryadav.github.io/Diwali-Store-Page/)

## 🧑‍💻 Technologies Used  
- **HTML5** – Semantic structure, image elements  
- **CSS3** – Custom styling, background image, padding, margin  
- **Bootstrap 4** – Flexbox layout for alignment  

## 📄 Features  
- Diwali top section with festive background  
- Product cards with image, name, and price  
- Cards arranged in rows using Bootstrap Flexbox  
- Clean spacing using padding and margin  
- "View More" button styled with Bootstrap  
- Mobile-friendly and responsive  

## 🎓 What I Learned  
- Approach to developing a layout structure  
- When to use HTML `<img>` vs CSS `background-image`  
- Adding Diwali top section with a background image  
- Adding product (bottle) section and individual card items  
- Aligning card items side by side using Bootstrap Flexbox  
- Breaking the layout into rows and adding multiple rows  
- Difference between CSS padding and margin — when to use each
